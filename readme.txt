== Description ==

Add notifications for new posts to a site with the Mailgun Subscriptions plugin.

== Installation ==

1. First install and activate the Mailgun Subscriptions plugin.
1. Then install and activate this plugin just as a normal WordPress plugin.
1. You'll find the "Mailgun Lists" settings page in the Settings admin menu. After setting up your Mailgun API settings, you can select a list that will receive notifications of new posts.
